# 心灵鸡汤 - 随机名言生成器

随机名言生成器，支持收藏和笔记功能。

## ✨ 功能特性

- 🎲 随机名言生成
- ❤️ 名言收藏管理
- 📝 个性化笔记记录


## 🚀 快速开始

1. 用浏览器打开 `index.html`
2. 点击"随机名言"按钮获取名言
3. 收藏喜欢的名言
4. 添加个人笔记

## 🛠 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
